## Reeds bekende kansen (uit eerder onderzoek — startvoorraad)
Niet door de loop gevonden, maar wel bruikbaar als vertrekpunt (zie `STRATEGIE.md`/`GBP-PLAN.md`/`PRIORITEIT.md`):

- **"goten reinigen [plaats]"** — bijna onbezet als dakdekker-term (alleen glazenwassers), jouw opener + upsell-wig. → matrix Tier A/B.
- **"daklekkage repareren / opsporen [plaats]"** — hoge CPC (€5+), lage comp, acute intentie. → matrix + artikel.
- **"verstopte regenpijp" (720, Laag)** — probleemterm, lage comp. → artikel + goten-pagina.
- **"stormschade dak [plaats]"** — piekt na elke storm, concurrenten reageren traag. → Tier C + storm-post.
- **Concurrenten op verkeerd review-platform** (Trustoo/Werkspot i.p.v. Google) → Google-review-voorsprong. → GBP-content.
- **Product/DIY-volume NEGEREN** (gootdrains, velux, dakkapel, dakisolatie) — hoog volume, geen huur-intentie. Geen pagina's voor maken.
